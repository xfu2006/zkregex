Primality proof for n = 237073:

Take b = 2.

b^(n-1) mod n = 1.

449 is prime.
b^((n-1)/449)-1 mod n = 40086, which is a unit, inverse 95022.

11 is prime.
b^((n-1)/11)-1 mod n = 91627, which is a unit, inverse 229329.

(11 * 449) divides n-1.

(11 * 449)^2 > n.

n is prime by Pocklington's theorem.
