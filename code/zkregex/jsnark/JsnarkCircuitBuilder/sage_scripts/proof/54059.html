Primality proof for n = 54059:

Take b = 2.

b^(n-1) mod n = 1.

179 is prime.
b^((n-1)/179)-1 mod n = 20388, which is a unit, inverse 9039.

151 is prime.
b^((n-1)/151)-1 mod n = 2252, which is a unit, inverse 34399.

(151 * 179) divides n-1.

(151 * 179)^2 > n.

n is prime by Pocklington's theorem.
