Primality proof for n = 84520951:

Take b = 2.

b^(n-1) mod n = 1.

157 is prime.
b^((n-1)/157)-1 mod n = 50142731, which is a unit, inverse 11786516.

97 is prime.
b^((n-1)/97)-1 mod n = 56685607, which is a unit, inverse 1423563.

(97 * 157) divides n-1.

(97 * 157)^2 > n.

n is prime by Pocklington's theorem.
