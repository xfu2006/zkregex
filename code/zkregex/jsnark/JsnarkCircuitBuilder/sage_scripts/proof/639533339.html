Primality proof for n = 639533339:

Take b = 2.

b^(n-1) mod n = 1.

1637 is prime.
b^((n-1)/1637)-1 mod n = 248876994, which is a unit, inverse 296391531.

853 is prime.
b^((n-1)/853)-1 mod n = 256057580, which is a unit, inverse 303978671.

(853 * 1637) divides n-1.

(853 * 1637)^2 > n.

n is prime by Pocklington's theorem.
