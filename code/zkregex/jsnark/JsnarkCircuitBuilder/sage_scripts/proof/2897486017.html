Primality proof for n = 2897486017:

Take b = 2.

b^(n-1) mod n = 1.

1069 is prime.
b^((n-1)/1069)-1 mod n = 1165776232, which is a unit, inverse 13801737.

743 is prime.
b^((n-1)/743)-1 mod n = 614414737, which is a unit, inverse 1707231904.

(743 * 1069) divides n-1.

(743 * 1069)^2 > n.

n is prime by Pocklington's theorem.
