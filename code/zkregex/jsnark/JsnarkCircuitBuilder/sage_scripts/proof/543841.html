Primality proof for n = 543841:

Take b = 2.

b^(n-1) mod n = 1.

103 is prime.
b^((n-1)/103)-1 mod n = 317586, which is a unit, inverse 524554.

11 is prime.
b^((n-1)/11)-1 mod n = 463383, which is a unit, inverse 256320.

(11 * 103) divides n-1.

(11 * 103)^2 > n.

n is prime by Pocklington's theorem.
