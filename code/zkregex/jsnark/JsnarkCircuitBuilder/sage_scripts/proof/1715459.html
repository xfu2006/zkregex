Primality proof for n = 1715459:

Take b = 2.

b^(n-1) mod n = 1.

967 is prime.
b^((n-1)/967)-1 mod n = 584615, which is a unit, inverse 1465867.

887 is prime.
b^((n-1)/887)-1 mod n = 1274704, which is a unit, inverse 679244.

(887 * 967) divides n-1.

(887 * 967)^2 > n.

n is prime by Pocklington's theorem.
