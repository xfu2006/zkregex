Primality proof for n = 496303:

Take b = 2.

b^(n-1) mod n = 1.

457 is prime.
b^((n-1)/457)-1 mod n = 45734, which is a unit, inverse 329476.

181 is prime.
b^((n-1)/181)-1 mod n = 180278, which is a unit, inverse 88247.

(181 * 457) divides n-1.

(181 * 457)^2 > n.

n is prime by Pocklington's theorem.
