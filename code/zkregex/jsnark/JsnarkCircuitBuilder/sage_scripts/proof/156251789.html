Primality proof for n = 156251789:

Take b = 2.

b^(n-1) mod n = 1.

137 is prime.
b^((n-1)/137)-1 mod n = 113755927, which is a unit, inverse 72467242.

23 is prime.
b^((n-1)/23)-1 mod n = 16839317, which is a unit, inverse 137727917.

(23^2 * 137) divides n-1.

(23^2 * 137)^2 > n.

n is prime by Pocklington's theorem.
