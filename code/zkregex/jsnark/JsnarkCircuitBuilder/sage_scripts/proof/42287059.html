Primality proof for n = 42287059:

Take b = 2.

b^(n-1) mod n = 1.

739 is prime.
b^((n-1)/739)-1 mod n = 12274859, which is a unit, inverse 20254790.

17 is prime.
b^((n-1)/17)-1 mod n = 36794259, which is a unit, inverse 18554779.

(17^2 * 739) divides n-1.

(17^2 * 739)^2 > n.

n is prime by Pocklington's theorem.
