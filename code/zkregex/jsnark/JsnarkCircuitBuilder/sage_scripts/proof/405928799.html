Primality proof for n = 405928799:

Take b = 2.

b^(n-1) mod n = 1.

4999 is prime.
b^((n-1)/4999)-1 mod n = 357727240, which is a unit, inverse 173101550.

3691 is prime.
b^((n-1)/3691)-1 mod n = 122990329, which is a unit, inverse 170220075.

(3691 * 4999) divides n-1.

(3691 * 4999)^2 > n.

n is prime by Pocklington's theorem.
