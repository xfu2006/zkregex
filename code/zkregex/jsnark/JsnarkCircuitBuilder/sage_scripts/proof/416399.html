Primality proof for n = 416399:

Take b = 2.

b^(n-1) mod n = 1.

331 is prime.
b^((n-1)/331)-1 mod n = 395953, which is a unit, inverse 16313.

37 is prime.
b^((n-1)/37)-1 mod n = 329629, which is a unit, inverse 370037.

(37 * 331) divides n-1.

(37 * 331)^2 > n.

n is prime by Pocklington's theorem.
